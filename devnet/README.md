# ShadowBase Devnet Configuration

> Privacy-Native Base L2 devnet settling to **Ethereum Sepolia** as L1.

## Architecture

```
┌─────────────────────────────────────────────────┐
│  ShadowBase L2 Devnet (Chain ID: 845311)        │
│                                                  │
│  op-geth (execution)  ←→  op-node (consensus)   │
│   :8545 HTTP RPC           :9545 RPC             │
│   :8546 WebSocket          :7300 Metrics         │
│   :8551 Engine API                               │
│                                                  │
│  op-batcher → batches txs to Sepolia L1          │
│  op-proposer → proposes output roots to L1       │
└──────────────────────┬──────────────────────────┘
                       │ settles to
                       ▼
┌─────────────────────────────────────────────────┐
│  Ethereum Sepolia (Chain ID: 11155111)           │
│  L1 Contracts: OptimismPortal, CDM, Bridge, etc. │
│  RAILGUN: 0xecfcf3b4ec647c4ca6d49108b311b7a7c9543fea │
└─────────────────────────────────────────────────┘
```

## Setup Flow

### Step 0: Prerequisites

```bash
# Install Go 1.21+
go version

# Install Foundry
curl -L https://foundry.paradigm.xyz | bash
foundryup

# Get a Sepolia RPC URL (Alchemy, Infura, or public)
export SEPOLIA_RPC_URL="https://eth-sepolia.g.alchemy.com/v2/YOUR_KEY"

# Fund your deployer account with Sepolia ETH from faucets
```

### Step 1: Build Binaries

```bash
# Build op-geth
cd ../op-geth
make geth

# Build op-node
cd ../optimism/op-node
make op-node
```

### Step 2: Generate JWT Secret

```bash
cd devnet/
openssl rand -hex 32 > jwt-secret.txt
```

### Step 3: Deploy L1 Contracts to Sepolia

This happens in the next plan phase. Once L1 contracts are deployed, update the placeholder addresses in `deploy-config/shadowbase-devnet.json`:
- `l1StandardBridgeProxy`
- `l1CrossDomainMessengerProxy`
- `l1ERC721BridgeProxy`
- `systemConfigProxy`
- `optimismPortalProxy`

### Step 4: Generate Genesis & Rollup Config

```bash
# From the optimism monorepo, use op-chain-ops to generate genesis
cd ../optimism

# This generates genesis.json and rollup.json from the deploy config
# Exact command depends on the tooling — see op-chain-ops docs
```

### Step 5: Start the Devnet

**Option A: Local binaries**
```bash
cd devnet/
chmod +x start-devnet.sh
SEPOLIA_RPC_URL="https://eth-sepolia.g.alchemy.com/v2/YOUR_KEY" ./start-devnet.sh
```

**Option B: Docker Compose**
```bash
cd devnet/
# Set your Sepolia RPC in .env.op-node first
docker compose up
```

### Step 6: Verify

```bash
# Check op-geth is running
curl -s http://localhost:8545 -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_chainId","params":[],"id":1}'
# Should return: {"jsonrpc":"2.0","id":1,"result":"0xce6bf"}  (845311 in hex)

# Check op-node is running
curl -s http://localhost:9545 -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"optimism_syncStatus","params":[],"id":1}'
```

### Step 7: Connect MetaMask

Add custom network in MetaMask:
- **Network Name:** ShadowBase Devnet
- **RPC URL:** http://localhost:8545
- **Chain ID:** 845311
- **Currency Symbol:** ETH
- **Block Explorer:** (leave blank or point to mini explorer later)
